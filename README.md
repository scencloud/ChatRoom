# ChatRoom
一个匿名聊天室开放源码

```markdown
# 💬 匿名聊天室 v1.0

一款开箱即用的内网匿名聊天室系统，单文件、零配置、双击即运行。

> 当前版本：v1.0.0 · 开源免费版
> 最新版本：v2.x（闭源，持续更新中）


## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🏠 多房间隔离 | 每个房间独立存在，数据完全隔离，互不干扰 |
| 🔑 双重验证 | 房间号 + 邀请码，确保只有授权用户进入 |
| 💬 实时聊天 | WebSocket 实时通信，消息秒收秒发 |
| 👤 自定义昵称 | 2-30字，支持中文、符号、表情，房间内唯一 |
| 🖼️ 图片上传 | 支持 JPG/PNG/GIF/WebP，≤3MB自动显示，>3MB点击加载 |
| 🎬 视频上传 | 支持 MP4/WebM/MOV，点击播放 |
| 📄 文件上传 | 支持任意格式文件，点击下载（下载前弹窗安全警告） |
| 📜 历史消息 | 新用户进入自动加载最近200条（后台可调） |
| 👥 在线列表 | 实时显示在线人数，进入/退出即时广播 |
| 🛡️ 防撞库 | 同一IP错误6次，封禁10分钟 |
| 🚦 频率限制 | 每人每秒最多3条消息，防刷屏 |
| 📊 管理后台 | 创建/删除房间，全局设置，修改密码 |
| 📋 访问日志 | 记录IP/路径/状态码/User-Agent，支持CF真实IP |


## 🚀 快速开始

### 直接运行（exe）

1. 下载 `ChatRoom.exe`
2. 双击运行
3. 复制控制台显示的**管理员初始密码**
4. 浏览器访问 `http://127.0.0.1:3986`
5. 后台管理：`http://127.0.0.1:3986/yesadmin`

### 源码运行

```bash
# 安装依赖
pip install flask flask-socketio eventlet werkzeug pillow

# 运行
python main.py

```

📁 目录结构

```
chatroom/
├── main.py              # 全部代码（单文件）
├── data/                 #自动创建
│   ├── chatroom.db      # 主数据库（房间/消息/设置）
│   └── access_logs.db   # 访问日志数据库
└── uploads/             # 上传文件存储(自动创建)
```

🖥️ 管理后台

功能 说明
创建房间 自定义房间号（2-20位字母数字）、邀请码（4-12位字母数字）、最大人数（最高500）
房间管理 查看所有房间的在线人数、消息总数、创建时间
删除房间 一键删除，自动清空所有消息和上传文件，踢出所有在线用户
全局设置 默认最大人数、历史消息加载条数、图片/视频/文件大小限制
修改密码 需验证旧密码
访问日志 查看IP、路径、状态码、User-Agent，支持手动加载和清空

🔒 安全机制

· ✅ SQL注入防护（参数化查询）
· ✅ XSS防护（前端 textContent 渲染）
· ✅ 文件类型验证（PIL 验证图片真实格式）
· ✅ 文件大小限制（图片/视频/文件独立可调）
· ✅ 防撞库（6次错误封禁10分钟）
· ✅ 消息频率限制（每秒3条）
· ✅ Session 24小时自动过期
· ✅ 上传文件 UUID 重命名
· ✅ 支持 Cloudflare 真实 IP（X-Forwarded-For）
